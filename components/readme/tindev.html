TinDev

Pre requirements   |    Technologies   |    How To Use   |   

Um tinder de desenvolvedores para caso ocorra um match será adiconado na sua lista de matchs, exibindo as informações do programador. Com o intuito de realizar par programming.

Pre requirements

  • Node.js
  • Yarn

How To Use

To clone and run this application, you'll need Git, Node.js v10.16 or higher + Yarn v1.13 or higher installed on your computer. From your command line: 

   # Clone this repository
$ git clone 
https://github.com/mesquini/tindev
> cd backend
> npm i
> yarn dev 

 ## after open other terminal:

> cd frontend
> npm i
> yarn start
