Rocketshoes

Tecnologias | Pré-requisitos | Instalação e execução

Projeto desenvolvido no GoStack 10, com o proposito de aprender sobre Redux e ReactJS. Basicamente é um E-commerce de tenis, aonde você tem duas telas, o de produtos e do carrinho.

🚀 Tecnologias

  • ReactJS
  • Redux
  • Redux-Saga
  • React Router v4
  • Axios
  • History
  • Immer
  • Polished
  • React-Toastify
  • styled-components
  • React-Icons
  • react-loading-skeleton
  • react-loader-spinner
  • json-server
  • Reactotron

✋🏻 Pré-requisitos

  • Node.js
  • Yarn
  • VS Code

Instalação e execução

Para clonar o repositorio, você precisará do Git, Node.js v10.16 + Yarn v1.13 

   # Clone esse repositorio
    > git clone https://github.com/mesquini/rocketshoes
    > cd rocketshoes
    > yarn
    ## Run API fake:
    > json-server db.json -p 3333 -w
    ## Abra outro terminal:
    > yarn start
