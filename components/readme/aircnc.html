AirCnC - Code and Coffee

Tecnologias | Pré-requisitos | Instalação e execução

Projeto realizado na Semana Omnistack 9, com o objetivo de oferecer/alugar Spots (espaços aonde a pessoa irá ficar) para programadores. Aonde ele pode procurar um Spot em alguma empresa do seu interessa, para realizar networks.

🚀 Tecnologias

  • ⚡ Express
  • 💾 Mongoose
  • ⌚️ Socket.io
  • 💻 Axios

✋🏻 Pré-requisitos

  • Node.js
  • Yarn
  • VS Code

Instalação e execução

Para clonar o repositorio, você precisará do Git, Node.js v10.16 + Yarn v1.13 

   # Clone esse repositorio
    > git clone https://github.com/mesquini/aircnc
    > cd backend
    > yarn
    > yarn dev 

    ## Antes abre outro terminal:

    > cd frontend
    > yarn
    > yarn start
