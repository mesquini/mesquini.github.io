Gerenciador de biblioteca

Um simples programa gerenciador de livros. Contendo as seguintes ações de cadastrar, editar, ver e deletar os livros, autores e editoras, fazendo o relacionamento aonde livro tem uma editora e um ou mais autores.

🚀 Tecnologias

  • Java - Linguage used
  • Postgre - SQL
  • Windonws Builder

✋🏻 Pré-requisitos

  • Eclipse
  • Postgre

🔥 Instalação e execução

  • Você precisa ter um banco, local ou online, que seja o Postgre;
  • Depois que tiver o banco, popular a tabela com os dados que está na pasta livraria;
  • Ao abrir o projeto, adicionar a lib postgresql-42.2.8.jar do Postgre;
  • Executar o projeto, o arquivo Main é o arquivo TelaInicial, que fica em src/View/TelaInicial;
  • Divirta-se com o programinha 😉
