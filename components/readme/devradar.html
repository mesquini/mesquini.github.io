DevRadar

Tecnologias | Pré-requisitos | Como usar

Projeto com o objetivo de fazer cadastro de programadores com seu username do GitHub na versão WEB E para utilizar o Radar, existe o APP aonde você filtra por tecnologia para entrar um Dev que programa naquela linguagem.

🚀 Tecnologias

Esse projeto foi desenvolvido na Semana Omnistask 10, utilizamos as seguintes tecnologias:

  • ReactJS
  • Axios
  • Cors
  • Socket.io
  • Material Icons
  • Mongoose
  • Express
  • VS Code

✋🏻 Pré-requisitos

  • Node.js
  • Yarn
  • VS Code

Como usar: 

  
    > git clone https://github.com/mesquini/semanaomnistack10
    > cd backend
    > yarn
    > yarn dev

    > cd frontend
    > yarn
    > yarn start
