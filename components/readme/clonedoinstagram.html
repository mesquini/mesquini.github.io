Clone Instagram

Tecnologias | Pré-requisitos | Como Usar

Projeto com a funcionalidade de criar posts, identido do Instagram, mostrar no feed e curtir uma foto ilimitadamente.

🚀 Tecnologias:

Esse projeto foi desenvolvido na Semana Omnistack 7, com as seguintes tecnlogias:

  • ReactJS
  • Axios
  • Cors
  • Socket.io
  • Mongoose
  • Express
  • VS Code

✋🏻 Pré-requisitos

  • Node.js
  • Yarn
  • VS Code

Como usar: 

   # Clone esse repositorio
    > git clone https://github.com/mesquini/clone_instagram
    > cd backend
    > yarn
    > yarn dev 

    ## Antes abre outro terminal:

    > cd frontend
    > yarn
    > yarn start
